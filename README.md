# Bash script for automatic installation proxy server based on Squid3

Script have 6 income parameters: 

1)SERVER USER;

2)SERVER_IP;

3)SERVER PASSWORD;

4)PROXY_PORT;

5)PROXY_USER_NAME;

6)PROXY_PASSWORD;

Example for use ./squid3_debian.sh root 10.10.10.10 ygW65erGDREsh0 6128 squiduser squidpass
